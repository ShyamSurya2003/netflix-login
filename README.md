# 🎬 Netflix Login Clone (Full Stack)

A full-stack Netflix-style login system built using **React (Vite)** for frontend and **Node.js + Express** for backend.

This project demonstrates authentication flow, API integration, and UI design inspired by Netflix.

---

## 🚀 Features

- 🔴 Netflix-style Login UI (Tailwind CSS)
- 🔐 User Signup & Login functionality
- 🔗 Frontend ↔ Backend API integration
- ⚠️ Error handling for invalid login/signup
- 🎯 Dashboard after successful login
- 🔄 Navigation between Login & Signup

---

## 🛠️ Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS

### Backend
- Node.js
- Express.js
- CORS
